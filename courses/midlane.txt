Regles de base : 
 
- Tag mid ne pas tag d'autre roles 
 
- Jouer grand max 4 champions ( syndra, arhi , kassadin ,yone ) 
 
- Max 5 games par jours sur ton main compte 
       - (prendre 5min de pause entre les games )
 
- Prendre l'habitude de jouer sans la musique pour aider a mieux focus 
 
- regle toujours ma faute : Toujours te dire que c'est de ta faute peut importe la situation  
 
- Désactiver le chat 
 
 
Fixer des objectif quand tu entre in game ( pourquoi tu lances la game ): 
 
- Si tu lances pour win des LP dans ce cas tag sur ton main compte et applique les regles de base 
 
- Si tu lances pour juste jouer lance des games sur un smurf et joue a ta guise 
 
 
 
 
Objectif in game ( spot tes erreurs ) :
 
exemple mauvais csing : Objectif de la game avoir 100 cs a 10min ect...
 
 
Review tes games pour analyser tes erreurs et te fixer des objectifs pour les prochaines games 
- te concentrer sur tes erreurs a toi  et review principalement la lanning phase 
( beaucoup regarder de gameplay de bon joueur twitch / youtube ) 
 
 
 
In game : 
- Connaitre son matchup ( tres important ) 
- Savoir quoi faire des premieres waves et se fixer un plan de jeu court termes 
Comment se fixer un plan de jeu court termes : 
           - Analyser les 2 compo et en deduire les possibilité ( exemple jungle passif en face et possibilité de lane bully dans ce cas hard pressure l'opposant ...)
- Analyser les forces et faiblesse de ton champion et jouer dessus le plus possible 
- Respecter les timing du jungle ( possibilité de gank ou non ) 
- jouer sur les waves ( les waves te donne les timing pour jouer et non l'invesre ) toujours regarder a ne pas perdre de wave 
- Priorisé un bon csing au reste
